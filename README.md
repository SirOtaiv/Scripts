## 🚀 Como usar

Cada diretório possui seu próprio arquivo de instruções, mas a lógica geral é:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/my-toolbox.git](https://github.com/seu-usuario/my-toolbox.git)
    ```
2.  **Dê permissão de execução (para scripts):**
    ```bash
    chmod +x ./automation/meu-script.sh
    ```
3.  **Execute:**
    ```bash
    ./automation/meu-script.sh
    ```

## 🔧 Destaques (Cool Stuff)

* **`backup_sync.sh`**: Sincroniza pastas locais com a nuvem via rsync.
* **`image_optimizer.py`**: Redimensiona e otimiza imagens em lote mantendo a qualidade.
* **`clean_docker.sh`**: Remove containers, imagens e volumes órfãos com um comando.
